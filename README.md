# Responsible AI Dashboard

This project demonstrates a simple Responsible AI monitoring dashboard.

The application trains a Decision Tree classifier on the Adult Income dataset
and visualizes model performance and fairness metrics.

The dashboard reports:

- Accuracy
- Precision
- Recall
- Group fairness comparison

The goal is to illustrate how AI systems can be monitored in accordance with
Responsible AI principles such as the NIST AI Risk Management Framework
and the transparency obligations of the EU AI Act.

## Run locally

streamlit run app.py
